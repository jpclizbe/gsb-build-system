/extra:  Extra GNOME SlackBuild package directory

Here are some extra/alternate packages that might be handy.

These can be installed using "installpkg".

Available packages
-----------------

mozilla-firefox  -
        Mozilla Firefox 3.0.3, built against XULRunner 1.9.0.1.  We're offering a mozilla
	package in extra/ alongside the official Slackware 12.1 version which is still
	the older 2.0.0.x series.

pidgin - 
	The latest stable pidgin package, built with support for pulseaudio.
