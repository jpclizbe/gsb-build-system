Development - helping the project

Information for people wanting to work on the GSB source. Patches, comments, suggestions welcome.

Getting the code from Subversion:

$ svn co http://svn.gnomeslackbuild.org/gsb/trunk gsb

$ svn co http://svn.gnomeslackbuild.org/gsb/branches 2.20.x

'trunk' holds the latest sources (may or may not be stable). GNOME stable (even numbered) version branches are always stable.

You can also browse the SVN source tree via a web browser:

<http://svn.gnomeslackbuild.org/viewvc/gsb/>

If you are a developer working on the GSB project, you are encouraged to subscribe to two important mailing lists:

  1. GSB Development Discussion List
  2. GSB SVN Commit List

Information about using subversion can be found in the Subversion Book.
