Installation Instructions

Installation via `slapt-get`

First, make certain you have the FRG package respository in your /etc/slapt-get/slapt-getrc;

SOURCE=http://mirror.datapipe.net/norlug/frg/frg-<version>

(don't forget to run `slapt-get --update` after adding new respositories!)

Then run the following command and grab a cup of coffee;

# slapt-get --search [0-9]frg$|awk '{print $1}'|xargs \
    -r slapt-get --install -y

Then, enjoy your new GNOME-enabled Slackware machine :)

Installation via ISO:

  1. Download the iso and md5 sum file: FRG ISO, then check the iso;

    $ md5sum -c file.iso.md5

  2. mount the iso;

    $ mkdir /mnt/iso

    $ mount -t iso9660 file.iso /mnt/iso -o loop

    $ cd /mnt/iso

  3. Then run either;

    $ ./min_install.sh

    or

    $ ./full_install.sh

    or manually use;

    $ upgradepkg --install-new

Uninstall Instructions

Please note that when uninstalling, you will have to reinstall some packages from the l disk set on your slackware cdrom. Please see replaced_slack_packs.txt for more infomation

Instructions are crude, but they do work.

  1. cd /var/log/packages
  2. ls -1| grep frg$| xargs -n 1 removepkg
