Installation Instructions

Installation via `slapt-get`

First, make certain you have the FRG package respository in your /etc/slapt-get/slapt-getrc;

  1. SOURCE=<full-path-to-the-mirror-of-your-choice>/frg-<version>

    See the Downloads section for a list of avaliable mirrors.

    (don't forget to run `slapt-get --update` after adding new respositories!)

  2. Then run the following command and grab a cup of coffee; 
    $ slapt-get --search [0-9]frg$|awk '{print $1}'|xargs \
    -r slapt-get --install -y

    Then, enjoy your new GNOME-enabled Slackware machine :)

Installation via ISO:

  1. Download the iso and md5 sum file: FRG ISO, then check the iso; 
    $ md5sum -c file.iso.md5
  2. mount the iso; 
    $ mkdir /mnt/iso
$ mount -t iso9660 file.iso /mnt/iso -o loop
$ cd /mnt/iso
  3. Then run either; 
    $ ./min_install.sh
    or 
    $ ./full_install.sh
    or manually use; 
    $ upgradepkg --install-new

Uninstall Instructions

Please note that when uninstalling, you will have to reinstall some packages from the "l" disk set on your Slackware CD-ROM. Please refer to replaced_slack_packs.txt for more information.

Instructions are crude, but they do work.

$ cd /var/log/packages
$ ls -1| grep frg$| xargs -n 1 removepkg
