About GSB: GNOME.SlackBuild

GSB is comprised of two projects, GSB and FRG.
Don't you think the world needs more acronyms? ;-)

  1. FRG: Freerock GNOME - a complete GNOME distribution for Slackware Linux. Encompasses all needed packages for a complete, clean installation of GNOME for Slackware. These packages are built using GSB (Gnome.SlackBuild).
  2. GSB: GNOME.SlackBuild - a collection of scripts that build a GNOME release on Slackware Linux.

The goal of this project is to run one script: GSB.SlackBuild and have it output Slackware packages for a GNOME release.

FRG is a secondary project built on top of GSB.
