About GSB: GNOME.SlackBuild

GSB is comprised of two projects, GSB and FRG.
Don't you think the world needs more acronyms? ;-)

  1. GSB: GNOME.SlackBuild - a collection of scripts that build a GNOME release on Slackware Linux.
  2. FRG: Freerock GNOME - GNOME packages for Slackware linux, these packages are built using GSB.

The goal of this project is to run one script: GSB.SlackBuild and have it output Slackware packages for a GNOME release.

FRG is a secondary project built on top of GSB.
