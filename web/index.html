GSB: GNOME.SlackBuild

Building a GNOME Distribution for the Slackware Linux Operating System.

The goal of this project is to run one script: GNOME.SlackBuild and have it output Slackware packages for a GNOME release.
The current gsb release 0.0.1 , builds GNOME version 2.9.90.

Binary packages will be available for the 2.9.91 release due out soon.
